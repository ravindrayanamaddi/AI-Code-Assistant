Example 1:
Task: Check if a number is prime
Code:
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True
Explanation:
A prime number has only two divisors: 1 and itself.
Test Cases:
assert is_prime(7) == True
assert is_prime(10) == False

Example 7:
Task: Fibonacci sequence
Code:
def fibonacci(n):
    a, b = 0, 1
    seq = []
    for _ in range(n):
        seq.append(a)
        a, b = b, a+b
    return seq
Explanation:
Each number is sum of previous two.
Test Cases:
assert fibonacci(5) == [0,1,1,2,3]

Example 20:
Task: Binary search
Code:
def binary_search(arr, x):
    left, right = 0, len(arr)-1
    while left <= right:
        mid = (left+right)//2
        if arr[mid] == x:
            return mid
        elif arr[mid] < x:
            left = mid+1
        else:
            right = mid-1
    return -1
Explanation:
Binary search works on sorted arrays.
Test Cases:
assert binary_search([1,2,3,4],3) == 2
